household-app(家計簿アプリ)
====

家計簿アプリを作成しました。アカウントを作成し、予算、買ったものなどを登録することで、予算-支出の計算が行われるようになっています。また、登録された買ったものは、カテゴリーごとに検索し、そのカテゴリーの合計支出を求めることなどもできます。
アプリ使用の際にゲストユーザを作成してあるので、機能を確認したい方は、ゲストユーザをつかってみてください。
